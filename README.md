

## Installation

1. Install Java (JDK 17)
   Verify installation:

   ```
   java -version
   ```

2. Install Google Chrome (latest version)

3. Install VS Code with Java extensions
   (Extension Pack for Java)

4. Download or clone the project

   ```
   git clone <repository-url>
   ```

---

## How to Run

1. Open the project in VS Code

2. Navigate to:

   ```
   src/com/nisarga/FlipkartTest.java
   ```

3. Right-click on `FlipkartTest.java`

4. Click **Run Java**
